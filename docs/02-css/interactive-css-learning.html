🎯 Fundamentos de CSS

💡 ¿Qué es CSS?

CSS (Cascading Style Sheets) es el lenguaje que usamos para dar estilo y diseño a las páginas web. Controla colores, fuentes, espaciado, diseño y mucho más.

Sintaxis Básica de CSS

styles.css
/* Regla CSS básica */ h1 { color: #a855f7; font-size: 2.5em; text-align: center; text-transform: uppercase; letter-spacing: 2px; } /* Múltiples propiedades */ .caja { background: linear-gradient(135deg, #667eea, #764ba2); color: white; padding: 30px; border-radius: 15px; box-shadow: 0 10px 25px rgba(0, 0, 0, 0.2); margin: 20px 0; transition: transform 0.3s ease; } .caja:hover { transform: translateY(-5px) scale(1.02); box-shadow: 0 15px 35px rgba(0, 0, 0, 0.3); } /* Párrafos estilizados */ p { font-family: 'Georgia', serif; line-height: 1.8; color: #334155; margin-bottom: 15px; } /* Botón moderno */ .boton { background: #3b82f6; color: white; border: none; padding: 12px 30px; border-radius: 50px; font-weight: bold; cursor: pointer; display: inline-block; transition: all 0.3s ease; } .boton:hover { background: #2563eb; transform: translateY(-2px); box-shadow: 0 5px 15px rgba(59, 130, 246, 0.4); }

🎲 Selectores CSS

🎯 Tipos de Selectores

Los selectores determinan a qué elementos HTML se aplicarán los estilos. CSS ofrece múltiples formas de seleccionar elementos.

selectores.css
/* Selector de elemento */ p { color: #475569; } /* Selector de clase */ .destacado { background-color: #fef3c7; padding: 10px; border-left: 4px solid #f59e0b; } /* Selector de ID */ #unico { background: #ddd6fe; padding: 15px; border: 2px solid #a78bfa; border-radius: 8px; } /* Selector de atributo */ a[target="_blank"] { color: #3b82f6; text-decoration: none; font-weight: bold; } a[target="_blank"]:after { content: " ↗"; } /* Selector descendiente */ .contenedor p { margin-left: 20px; color: #6366f1; } /* Selector hijo directo */ .lista > li { background: #f0f9ff; margin: 5px 0; padding: 10px; list-style: none; } /* Selector de hermano adyacente */ h3 + p { font-size: 1.1em; color: #059669; font-weight: 500; } /* Selector de hermanos general */ h3 ~ p { margin-left: 10px; } /* Pseudo-clases */ button:hover { background: #ec4899; color: white; transform: scale(1.05); } input:focus { outline: 3px solid #a855f7; border-color: #a855f7; } li:nth-child(odd) { background: #fef3c7; } li:nth-child(even) { background: #dbeafe; } /* Pseudo-elementos */ .quote::before { content: "❝"; font-size: 2em; color: #a855f7; } .quote::after { content: "❞"; font-size: 2em; color: #a855f7; } p::first-line { font-weight: bold; color: #1e293b; } /* Selector múltiple */ h1, h2, h3 { font-family: 'Arial', sans-serif; color: #1e293b; } /* Selector universal */ * { box-sizing: border-box; } /* Selector :not() */ .items li:not(.excluido) { border: 2px solid #10b981; padding: 8px; margin: 5px; }

💡 Especificidad CSS

La especificidad determina qué estilos se aplican cuando hay conflictos:

  • ID: 100 puntos (#id)
  • Clase: 10 puntos (.clase)
  • Elemento: 1 punto (p, div, etc.)
  • !important: Gana siempre (úsalo con moderación)

📦 Box Model

📏 El Modelo de Caja

Todos los elementos HTML son cajas rectangulares. El Box Model define cómo se calculan las dimensiones y espaciado de estos elementos.

/* Box Model Básico */ .box { /* Contenido */ width: 300px; height: 200px; /* Padding (espacio interno) */ padding: 20px; /* También puedes especificar cada lado */ /* padding-top: 10px; padding-right: 15px; padding-bottom: 10px; padding-left: 15px; */ /* O usar shorthand */ /* padding: 10px 15px; */ /* vertical horizontal */ /* padding: 10px 15px 20px 25px; */ /* top right bottom left */ /* Border */ border: 3px solid #6366f1; border-radius: 12px; /* Margin (espacio externo) */ margin: 20px auto; /* centrado horizontal */ /* Background */ background: linear-gradient(45deg, #a855f7, #ec4899); color: white; } /* Box-sizing */ .box-content { box-sizing: content-box; /* Default */ width: 200px; padding: 20px; border: 5px solid #3b82f6; /* Ancho total = 200 + 40 + 10 = 250px */ } .box-border { box-sizing: border-box; /* Incluye padding y border */ width: 200px; padding: 20px; border: 5px solid #10b981; /* Ancho total = 200px */ } /* Display types */ .inline { display: inline; background: #fef3c7; padding: 5px 10px; margin: 10px; /* margin vertical no funciona en inline */ } .inline-block { display: inline-block; background: #dbeafe; padding: 10px; margin: 10px; } .block { display: block; background: #fce7f3; padding: 15px; margin: 10px 0; } /* Overflow */ .overflow-demo { width: 200px; height: 100px; border: 2px solid #a855f7; padding: 10px; margin: 10px; } .overflow-visible { overflow: visible; /* Default */ background: #fef3c7; } .overflow-hidden { overflow: hidden; background: #dbeafe; } .overflow-scroll { overflow: scroll; background: #fce7f3; } .overflow-auto { overflow: auto; background: #dcfce7; } /* Min y Max dimensions */ .responsive-box { min-width: 200px; max-width: 500px; min-height: 100px; width: 80%; padding: 20px; background: #e0e7ff; border: 2px solid #6366f1; margin: 20px auto; }
